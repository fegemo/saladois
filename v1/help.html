 Os Cavalos de Tróia trabalham com um sistema de cliente e servidor (client and server). Quando baixar um desses programas da internet irá receber dentro de um arquivo ZIP o programa do Client e o programa do Server.

Client no caso do NetBus: É o programa chamado NetBus.exe, dentro dele voce encontrará varias opções diferentes como abrir drive de CD-rom, travar teclado, mudar mouse para canhoto, ver tela, ver arquivos, enviar arquivos, ver o que ele está escrevendo, informação do computador e mais algumas utilides que eu não lhe aconselho a utilizar como senha no computador da vitima e tal... esse programa é denominado Client porque é nele que vc se conecta na pessoa no qual essa pessoa possa é vítima.

Server no caso do NetBus: é um programa chamado Patch.exe que apos a pessoa o ter executado ele vai se auto-copiar para a pasta raiz de todo o computador (C:\windows\) hehehe, apos a pessoa ter executado o servidor seu computador estará aberto uma porta que tem o número de 12345 (esse é o numero oficial do NetBus, cada trojan tem o seu). Depois que a pessoa executar o Server é só abrir o programa client, colocar o IP do server e esta feita á festa! :-)

  1. Baixe o NetBus

  2. Mude o nome do arquivo server para qualque coisa que lhe venha a cabeça como protecaodetela.exe e manda pro carinha dizendo que é uma roteção de tela hiper legal, :-)

  3. Mande o server para o cobaia e peça para ele executar

  4. Quando ele disser que executou e não aconteceu nada você entra em ação com o Client.

  5. Coloque o IP dele e clique em "Connect"

  6. Boa diversão, uahuahauahu!!!!
